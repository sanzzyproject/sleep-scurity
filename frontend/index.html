Infrastruktur Sistem

Arsitektur Zero-Knowledge

Web aplikasi ini berjalan sepenuhnya di sisi klien (Client-Side Processing). Sistem kami tidak mengirimkan satu byte pun data visual ke server eksternal, memastikan privasi absolut.

Penyimpanan Terisolasi (IndexedDB)

Enkapsulasi data tersimpan di dalam memori internal browser perangkat keras Anda. Bukti otentik hanya dapat diakses melalui antarmuka mesin lokal yang Anda gunakan.

Background Execution

Setelah terpicu oleh anomali sentuhan, mesin akan menginisialisasi loop capture setiap 5 detik. Catatan: Rutinitas OS modern (iOS/Android) dapat menangguhkan proses latar belakang ini jika layar dalam mode Sleep untuk protokol efisiensi daya.
